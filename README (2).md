This project helps analyze student marks easily.  
It can take manual input or read from a CSV file, calculate average, median, highest, lowest marks, assign letter grades, and show pass/fail summary.

#### Features:
- Read marks manually or from CSV
- Calculate average, median, max, min
- Assign grades (Aâ€“F)
- Display pass/fail students
- Show formatted result table

#### How to Run:
1. Open terminal inside project folder.
2. Run: `python gradebook.py`
3. Choose 1 for manual input or 2 for CSV file.
